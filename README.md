# Movies info app (Flixx-APP)

Movie info application built with vanilla JavaScript that uses **[TMDB API](https://developers.themoviedb.org/3) version 3**. This app was built following [ Brad Traversy](https://www.traversymedia.com/) online course ModernJS.

---

## Live Demo (my version)

[Visit site here](https://rcd-flixxapp.netlify.app/)

---

## Tech Stack

- JavaScript(ES6) and FetchApi
- HTML5
- CSS3

---

## Features

- Asynchronous data fetching using `async / await`
- Dynamic rendering of movie data
- Clean UI built with CSS
- Responsive layout

---

## Getting Started (Local Setup)

- To run this app you need an api key from the [TNMDB](https://www.themoviedb.org/settings/api)

---

### Prerequisites

- API key from [TNMDB](https://www.themoviedb.org/settings/api)

### Installation

1. Clone the repository
2. Open `index.html` in your browser  
   _(or use a local server for best results)_
